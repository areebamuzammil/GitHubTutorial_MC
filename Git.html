Step-1: Create GitHub Account

First we need to create our GitHub account.For this Click here

Step-2: Create a new Repository on GitHub

To create a new repository on GitHub, log in and click on "+" on the top right corner of Navigation Bar. After that click on "New repository" from the drop down options.

Write a name for your repository and give brief description.Add a README file and click on "Create repository" button.

Our repository is created.
