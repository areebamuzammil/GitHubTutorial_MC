GitHub Tutorial for Beginners

Step-1: Create GitHub Account

First we need to create our GitHub account. For this Click here

Step-2: Create a new Repository on GitHub

To create a new repository on GitHub, log in and click on "+" on the top right corner of Navigation Bar. After that click on "New repository" from the drop down options.

Write a name for your repository and give brief description. Add a README file and click on "Create repository" button.

A new repository will be created.

Step-3: Clone the repository

Create a new file on the system. Open Command Prompt with the path of the folder in which we are storing our file in.

Copy the link of the repository by click on the "code"after opening the repository on GitHub. After that write

"git clone [our repository link] "

on Command Prompt.

By doing this we are copying our GitHub repository to our local system.

Step-4: Add file in repository

Move the newly created file in the repository folder, on our systems. After that write

"git add [our file name]"

on Command Prompt.

By doing this git selects that file, and moves it to the staging area, marking it for inclusion in the next commit. We can select all files, a directory, specific files, or even specific parts of a file for staging and commit. This means if we git add a deleted file the deletion is staged for commit.
